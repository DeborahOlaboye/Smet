# Developer Quickstart

1. Clone the repo
2. Run `scripts/setup-dev.sh` (Unix) or `scripts/setup-dev.ps1` (PowerShell)
3. Start Hardhat and deploy dev contracts: see `docs/setup.md`
4. Start frontend: `cd frontend && npm run dev`
5. Run unit tests: `cd frontend && npm test`

This file provides a short checklist for daily development tasks.
