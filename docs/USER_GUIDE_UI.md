# User Guide: Using the App UI

This page shows step-by-step UI actions for common flows.

## Open reward flow
1. Connect a wallet
2. Navigate to the rewards page
3. Choose a reward box and click `Open`
4. Confirm in your wallet
5. Wait for the transaction to confirm and view your prize

## View history
- Use the `My Activity` page to see past opens and outcomes

## Reporting issues
- If something looks incorrect, grab the tx hash and open an issue including the hash and screenshots.
