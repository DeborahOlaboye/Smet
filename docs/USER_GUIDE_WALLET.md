# User Guide: Wallets & Safety

This short guide helps users connect wallets safely and troubleshoot common wallet issues.

## Supported providers
- MetaMask
- WalletConnect

## Connecting
1. Click `Connect Wallet`
2. Choose your provider
3. Approve connection in the wallet UI

## Common issues
- No provider listed: install MetaMask or use WalletConnect
- Wrong network: switch wallet network to the one shown in the app

## Safety notes
- Only connect to trusted apps
- Review gas and transaction details before confirming

If you need help, open an issue using the `developer-onboarding` issue template with the `help wanted` label.
